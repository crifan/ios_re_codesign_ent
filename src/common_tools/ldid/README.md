# ldid

* ldid
  * 是什么：用于修改二进制的entitlement权限的工具
  * 作者：`saurik`
  * 作用
    * 查看和设置entitlement权限
    * 伪签名
      * 对于二进制签名，生成SHA1和SHA256的hash值，以便于iPhone能够正常执行二进制
  * Cydia中叫做：`Link Identity Editor`
